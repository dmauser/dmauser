## About

I focus on **Azure networking** — designing, testing, and documenting connectivity, routing, security,
and hybrid scenarios. This profile is a curated index of hands-on **labs, sample scripts, and reference
implementations** I maintain across multiple repositories. Most content is reproducible end-to-end so you
can deploy, break, and learn from real Azure topologies.

🔭 **Currently:** building **GitHub Copilot CLI extension packs** (Network Desk, Money Desk, Compute Desk,
Financial Services) and publishing reproducible Azure networking labs from real customer scenarios.

📫 **Connect:** [LinkedIn](https://www.linkedin.com/in/danmauser/) · [GitHub Gists](https://gist.github.com/dmauser) · [Azure Networking Blog (Tech Community)](https://techcommunity.microsoft.com/category/azure/blog/azurenetworkingblog)
